# APPLE-ISTORES

A GUI application for the **APPLE STORE**, designed for **cashiers**, **managers**, and **product & stock management**.  
The Manager can create new employee accounts, add products, and manage stock. The Cashier can only view and search products and stock.

---

## Features by Role

| Role      | Features                                                                 |
|-----------|--------------------------------------------------------------------------|
| **Manager** | Create employee accounts<br>Add products and stock<br>Update products and stock<br>Delete products and stock<br>Search products and stock<br>View all products and stock |
| **Cashier** | View products and stock<br>Search products and stock |
